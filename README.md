# 06. Neuro-Priority Radar

## Concept
A decision-making algorithm that ranks tasks based on a weighted scoring system. It solves the problem of "procrastination by confusion" by providing a mathematical output for what to do next.

## Innovation
This project implements **Lambda Functions** and the **Sorting Algorithm** (`sorted()`). It moves away from simple lists to "List of Dictionaries," showing a higher level of data structure organization while maintaining a beginner-friendly core.

## How to Run
1. Run: `python main.py`
2. Enter your tasks and rate their importance and urgency (1-10).
3. Type 'stop' to see the calculated priority list.
